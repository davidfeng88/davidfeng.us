## 缘起

我姓封。这个姓氏比较罕见，似乎我在工作生活中除了亲戚之外还没有遇到过其他姓封的[^feng]。前几天看[视频](https://www.youtube.com/watch?v=PR-uSiayUOI&t=500s)，里面提到唐代有本著名的笔记叫《封氏闻见记》。看到这个题目我就来精神了，这不是可以用作本网站的名字吗？[维基百科](https://zh.wikipedia.org/wiki/%E5%B0%81%E6%B0%8F%E9%97%BB%E8%A7%81%E8%AE%B0)上说作者是唐代的[封演](https://zh.wikipedia.org/wiki/%E5%B0%81%E6%BC%94)，纪晓岚在《四库全书总目提要》中评价：“唐人小说多涉荒怪，此书独语必徵实”。看来有点意思，找来看看。

## 节选和吐槽

全书篇幅不长，十卷，共101个条目，讲的东西天南地北，什么都有（更像本网站了）。可惜出了学校这么多年，文言文基本都还给老师了。找了几个AI试图翻译成白话文，似乎DeepSeek效果最好，果然“更懂中文”。下面节选了一些我觉得有意思的片段。作者生卒年不详，但已知756年进士，800年还有活动。他经历了安史之乱（755-763）,可以看到不少内容是有时代精神的。

#### 定谥

{{< tabs >}}

{{% tab title="白话文" %}}

太常博士掌管谥法。职事官三品以上去世，由旧部下整理行状，上报尚书省，考功司校勘，下发给太常博士拟议，然后上报省里，省里议定后，再报告皇帝。从前周公是文王的儿子，谥号为文公。只要有美德，不避同谥。谥号用两个字，一个字表示本质，一个字表示文采。或文或质，取决于当时礼官的意思，没有固定的规则。自汉魏以来，即使道德高尚的人，如果没有爵位，就不加谥号。晋代王导上疏，说武官有爵位的一定给谥号，很违背制度的本意。从此公卿没有爵位的也都给谥号。唐太宗时的郑公魏征，唐玄宗时的梁公姚崇、燕公张说、广平公宋璟、郇公韦安石，都谥为“文贞”二字。这是大臣中最美的谥号，不是德望特别高的人得不到这个谥号。有唐以来，五个人同用一个谥号，也不避嫌。……

{{% /tab %}}

{{% tab title="原文" %}}

这个我有疑问，因为以前听说过文官最高的谥号是“文正”，范仲淹和曾国藩都是这个谥号，怎么这里又说是“文贞”？没关系，都2026了，有问题就问AI嘛。DeepSeek说：宋仁宗赵祯即位后，为避其名讳（“祯”与“贞”同音），便将“文贞”改称为“文正”。此后，经北宋名臣司马光的推崇，将其定义为“谥之极美，无以复加”，认为“文”代表道德博闻，“正”代表靖共其位，是文人道德的极致，“文正”便从此被后世公认为文臣的最高谥号。在宋、元、明、清四朝，“文正”一直是文臣的顶级谥号，获得者极少。著名的有宋代的范仲淹、司马光，以及清代的曾国藩等。

#### 官衔

{{< tabs >}}

{{% tab title="白话文" %}}

“官衔”这个名称，大概起源于近代。当时吏部选官授职，必须保留资历。上报朝廷的时候，先把原有的官名品级写在前面，然后写上拟授的官职在后面。使新旧相连不断，所以叫做“官衔”，也叫“头衔”。之所以叫做“衔”，就像人嘴里含着东西，取连续不断的意思。又像马有衔铁，用来控制马头。前面的马已经走了，后面的马接续而来，前后不绝。古人称之为“衔尾相属”，就是这个意思。

{{% /tab %}}

{{% tab title="原文" %}}

官衔之名，盖兴近代。当时选曹补授，须存资历。闻奏之时，先具旧官名品于前，次书拟官于后。使新旧相衔不断，故曰官衔。亦曰头衔。所以名为衔者，如人口衔物，取其连续之意。又如马之有衔，以制其首。前马已去，后马续来，相次不绝者。古人谓之衔尾相属，即其义也。

#### 文宣王庙树

{{< tabs >}}

{{% tab title="白话文" %}}

{{% tab title="原文" %}}

……广德初，御史大夫李季卿河南宣慰，过曲阜，谒文宣王庙，因遍寻鲁中旧迹。县使一老人导引，每至一所，老人辄指云，此是颜子陋巷，此是鲁灵光殿阶，此是泮宫。季卿闻之皆沈吟嗟赏，曰：“此翁真鲁人也。”次至池水，复指之：“此是钓鱼池。”季卿问曰：“何人钓鱼？”老人对曰：“鲁人灵光此钓鱼。”季卿曰：“鲁人败矣。”又于路侧见古碑，季卿问是谁碑，诸君并不能对。有一尉遽走至碑下，仰读其题云“李君德政碑”，走还白云：“李君德政碑。”季卿笑曰：“此与鲁人灵光何异？”

{{% /tab %}}

{{< /tabs >}}

这个没看懂，果断扔给DeepSeek。首先，文宣王是唐玄宗在开元二十七年（公元739年）追封给孔子的谥号。孔老先生一辈子政治不得志，死后被各个帝王封圣封王，令人唏嘘。然后老人提到了几处景点的典故有：

- 颜子陋巷：指孔子弟子颜回所居住的简陋小巷。孔子曾赞扬颜回 “一箪食，一瓢饮，在陋巷，人不堪其忧，回也不改其乐” 。后来这个典故被用来比喻安贫乐道、品德高尚的君子，即使身处陋室，内心依然富足。
- 鲁灵光殿阶：鲁灵光殿是汉景帝之子鲁恭王刘余在曲阜修建的宫殿。西汉末年，战火纷飞，长安城中宏伟的未央宫、建章宫都毁于一旦，唯独远在鲁地的灵光殿岿然独存。因此，“灵光殿”成为一个文化符号，比喻历经劫难而硕果仅存的人或事物。
- 泮宫：西周时期，天子设立的大学称为“辟雍”，诸侯设立的大学则称为“泮宫”。它是诸侯国的最高学府，也是举行典礼、宴饮、习射的重要场所。后来，“泮宫”也泛指各地的学宫。曲阜作为鲁国故都，其“泮宫”遗址更是历史悠久的文化象征。

向导一开始提到上述典故及景点，李季卿印象深刻。但老人说灵光钓鱼池是一个叫灵光的人在此钓鱼，就搞错了。这个“灵光”不是人名，而是殿名。其实上面已经提到了“灵光殿阶”，可见向导业务不精，死记硬背，不求甚解。后面看到的“李君德政碑”，与李季卿无关，是路旁一块为某位李姓官员歌功颂德的古碑，向导和官员们都不知道是谁的。一个县尉跑过去，只读了碑额上“李君德政碑”几个大字就跑回来汇报，这其实和老人那个“鲁人灵光钓鱼”的笑话如出一辙——都是只知其表，不明其里。

#### 推让

{{% tab title="白话文" %}}

高利从濠州刺史调任楚州刺史。当时江淮一带米价昂贵，职田里收获的粳米，价值几千贯。淮地的惯例，接任者在五月五日以前到任的，可以领取当年的职田收入。高利想把这份收入让给前任，所以从濠州出发后，沿途故意耽搁停留。等到过了期限几天之后，才到达楚州。士人们都称赞他。

{{% /tab %}}

{{% tab title="原文" %}}

高利自濠州改为楚州。时江淮米贵，职田每得粳米，直数千贯。淮例，替人五月五日已前到者，得职田。利欲以让前人，发濠州，所在故为淹泊。候过限数日，然后到州。士子称焉。

#### 奇政

{{< tabs >}}

{{% tab title="白话文" %}}

李封担任延陵县令。官吏百姓犯了罪，他不施加杖刑，只是命令他们裹上绿色的头巾来羞辱他们。根据犯罪的轻重，规定戴头巾的天数，期满才释放。吴地的人戴上这种头巾，出入乡里，认为是非常大的耻辱，因此都互相劝诫勉励，没有人敢犯法。该县的赋税常常比其他县先完成。他离任时，竟然没有鞭打过一个人。

{{% /tab %}}

{{% tab title="原文" %}}

李封为延陵令。吏人有罪，不加杖罚，但令裹碧头巾以辱之。随所犯轻重，以日数为等级，日满乃释。吴人着此服，出人州乡，以为大耻，皆相劝励，无敢僭违。赋税常先诸县，去官，竟不捶一人。

#### 掩恶

{{< tabs >}}

{{% tab title="白话文" %}}

检校刑部郎中程皓，性格周密谨慎，不谈论别人的短处。每次在同僚中看到有人诋毁别人，他从不接话。等那人说完，他才慢慢替被诋毁的人分辩说：“这都是大家乱传的，其实不是那样。”并且说那人的好事。曾经有一次在宴席上被人恶毒辱骂，他始终没有怒色。他慢慢起身避开说：“那个人醉了，不值得和他计较。”他就是如此高雅持重。

{{% /tab %}}

{{% tab title="原文" %}}

检校刑部郎中程皓，性周慎，不谈人短。每于侪类中见人有所訾毁，未曾应对。候其言毕，徐为分雪之曰：“此皆众人妄传，其实不尔。”更说其人美事。曾坐被人酷骂，竟席无怒色。皓徐起避之曰：“彼人醉耳，何可与言。”其雅重如此。

### 卷十

#### 讽切

{{< tabs >}}

{{% tab title="白话文" %}}

贺知章任秘书监，多年没有升迁。张九龄被罢免宰相后，在朝中对贺知章说：“我张九龄多事，心里遗憾没能让您升官，为此很抱歉。”贺一向诙谐，应声答道：“我知章蒙受相公的庇护可不少。”张问：“有什么庇护？”贺说：“自从相公在朝堂上，没人敢骂我知章是‘獠’。您罢相以后，‘尔’、‘汝’这类单字称呼，才渐渐又出现了。”张九龄大为惭愧。

{{% /tab %}}

{{% tab title="原文" %}}

贺知章为秘书监，累年不迁。张九龄罢相，于朝中谓贺曰：“九龄多事，意不得与公迁转，以此为恨。”贺素诙谐，应声答曰：“知章蒙相公庇荫不少。”张曰：“有何相庇？”贺曰：“自相公在朝堂，无人敢骂知章作獠。罢相以来，尔汝单字，稍稍还动。”九龄大惭。

{{% /tab %}}

{{< /tabs >}}

DeepSeek又立功了，补充了关键背景：贺知章和张九龄都是南方人。贺知章是越州永兴（今浙江杭州萧山）人，张九龄是韶州曲江（今广东韶关）人。两人都来自南方，而唐代的政治文化中心在关中（长安、洛阳），北方人往往瞧不起南方人，常用“獠”（读lǎo，本指南方少数民族）来骂南方人，类似于“蛮子”“南蛮”。

#### 避忌

{{< tabs >}}

{{% tab title="白话文" %}}

兼御史大夫韦伦奉命出使吐蕃，派御史苟曾为判官。出发日期已定。有人对韦伦说：“吐蕃忌讳‘狗’字（‘苟’与‘狗’音近），大夫带一个姓苟的判官，怎么去求好？”韦伦赶紧上奏此事，皇帝下令把“苟”改为“荀”，但人还是那个人。等出使回来，苟曾就改姓荀，不再恢复原姓。

{{% /tab %}}

{{% tab title="原文" %}}

兼御史大夫韦伦，奉使吐蕃，以御史苟曾为判官，行有日矣。或谓伦曰：“吐蕃讳狗，大夫将一苟判官，何以求好？”伦遽奏其事，今上令改苟为荀，而其人不易。及使还，曾遂姓荀，不归旧姓。

#### 失误

{{< tabs >}}

{{% tab title="白话文" %}}

阳伯博任山南某县县丞，他的妻子陆氏是名门之女。县令的妻子姓伍。有一天，众官员的妻子聚会。见面后，县令夫人问县丞夫人姓什么，回答姓陆。又问主簿夫人姓什么，回答姓漆。县令夫人生气地走进里屋。其他夫人不知怎么回事，想回去。县令听说后赶紧进来，问妻子怎么回事，妻子说：“县丞夫人说姓陆，主簿夫人说姓漆，因为我姓伍（五），所以故意戏弄我。其他官员的夫人幸亏我没问，要是问了，一定说姓八、姓九。”县令大笑说：“人家各有各的姓，怎么是这样呢？”又让妻子出来。

{{% /tab %}}

{{% tab title="原文" %}}

阳伯博任山南一县丞，其妻陆氏，名家女也。县令妇姓伍也。他日，会诸官之妇。既相见，县令妇问赞府夫人何姓，答曰姓陆。次问主簿夫人何姓，答曰姓漆。县令妇勃然入内，诸夫人不知所以，欲却回。县令闻之遽入，问其妇，妇曰：“赞府妇云姓陆，主簿妇云姓漆，以吾姓伍，故相弄耳。余官妇赖吾不问，必曰姓八、姓九。”县令大笑曰：“人各有姓，何如此？”复令其妇出。
